## Login Application

MEAN login application (Node.js, Express, MongoDB, Angular 4, Bootstrap 3)

Version: 1.0

#### Usage:

Server: 
```
npm install
npm start
```

For development use: DEBUG=basic-login:* npm start devstart

Config is here: config/default.json

Client:
```
cd client
ng serve
```

#### API Endpoints:

```
POST api/register
POST api/auth
GET api/profile
```
